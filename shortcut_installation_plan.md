# Добавление Mini App на рабочий стол (Идея и План реализации)

Этот документ сохранен для последующего возвращения к задаче добавления ярлыка приложения на рабочий стол пользователя.

## Нативное решение через Telegram WebApp SDK

Telegram предоставляет официальный API для добавления Mini App на домашний экран мобильного устройства.

### 1. API Методы
- `window.Telegram.WebApp.checkHomeScreenStatus(callback)`
  - Проверяет текущий статус ярлыка.
  - Возвращает один из статусов: `added`, `missed`, `unsupported`, `unknown`.
- `window.Telegram.WebApp.addToHomeScreen()`
  - Вызывает диалоговое окно операционной системы/Telegram для добавления ярлыка.

---

## План реализации в интерфейсе

1. **Проверка при запуске**:
   - После успешного входа водителя (в `App.jsx` после `setUser`) вызвать `checkHomeScreenStatus`.
   - Если статус равен `missed`, показать аккуратную плашку или всплывающее окно (modal).
   - Если пользователь закрыл плашку, сохранить флаг в `localStorage` (например, `dismissed_shortcut_prompt: "true"`), чтобы не надоедать при каждом запуске.

2. **UI Компонент**:
   - Красивое модальное окно с кнопками:
     - **[ Установить ярлык ]** — запускает `window.Telegram.WebApp.addToHomeScreen()`.
     - **[ Как настроить разрещения ]** — открывает краткий иллюстрированный гайд.
     - **[ Позже ]** — закрывает окно.

## Инструкции по разрешениям для пользователей

### Android (Xiaomi, Huawei, Oppo, Realme и др.)
В некоторых прошивках Android по умолчанию отключено разрешение на создание ярлыков сторонними приложениями (в данном случае ярлык создает Telegram).

**Пошаговая инструкция для пользователя:**
1. Откройте **Настройки** телефона.
2. Перейдите в раздел **Приложения** (или Управление приложениями) → **Разрешения**.
3. Выберите **Другие разрешения** (или Специальные разрешения).
4. Найдите **Telegram** в списке.
5. Включите пункт **"Ярлыки рабочего стола"** (или "Добавление ярлыков").

### iOS (iPhone / iPad)
На устройствах Apple ярлыки поддерживаются нативно, если установлена свежая версия iOS и официальный клиент Telegram. Дополнительных системных разрешений настраивать не нужно, достаточно подтвердить действие во всплывающем окне Telegram.
